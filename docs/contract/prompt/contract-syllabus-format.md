## Output format (Middle School – Grammar)

For each item, use this shape:

**[Item N]**
[Stem: one or two sentences with one blank or one transformation instruction.]

① [option 1]
② [option 2]
③ [option 3]
④ [option 4]
⑤ [option 5]

(If the item is "write the correct form" or transformation, use a short blank line instead of ①–⑤.)
